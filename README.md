## Install

```
git clone https://github.com/neurotech/rosebank-canvas.git
cd rosebank-canvas
npm i
```

## Scripts

### dev

**`npm run dev`**

Uses `nodemon` to watch `./src/` for any changes and re-runs `build.js` accordingly.

### build

**`npm run build`**

Builds assets found in `./src/` to `./build/`
